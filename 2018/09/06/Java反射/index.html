    Java 动态加载类

    Class.forName(${classpath})

    编译时刻加载类是静态加载类；

    运行时刻加载类是动态加载类。

    静态加载类有着诸多不便之处，编译器必须将程序中所出现的所有类找到并编译成功，程序才可能正确运行。而动态加载类则不必这样“万事俱备”，只需要编译程序运行中用到的类， Class.forName(${classpath}) 这个静态方法通过类的全路径找到所需的类并加载，可以非常灵活地运行程序。
